# Aryan Singh | Portfolio

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- **React 19** - UI library
- **Vite** - Build tool and dev server
- **Tailwind CSS v4** - Utility-first styling
- **Framer Motion** - Animations
- **React Router** - Client-side routing
- **React Icons** - Icon library

## Features

- Dark themed, responsive design
- Animated page transitions and scroll-triggered effects
- Mobile-friendly hamburger navigation
- Rotating role text on the hero section
- Skill cards, project showcase, and contact form
- Links to GitHub, LinkedIn, and Email

## Projects Showcased

- **University Management System** - MERN stack app with role-based access control
- **Dockerized Static Website Deployment** - Containerized web app with Docker
- **Hotel Buddy** - AI-powered hotel search chatbot using Gemini API

## Getting Started

```bash
cd portfolio-react
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```
portfolio-react/
├── src/

│   ├── components/      # Navbar, Footer
│   ├── pages/           # Home, About, Projects, Contact
│   ├── App.jsx          # Router layout
│   ├── main.jsx         # Entry point
│   └── index.css        # Tailwind config and global styles

```
